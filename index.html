< ABOUT ME />

I had never imagined that I would become a Full Stack Web Developer if you would have asked me where I see myself working in the future. My love with this world came when I did a beginner HTML and CSS course in an Adult School in London. I enjoyed some much that I decided to make a Master in Computing for Financial Markets at University of Stirling to learn about programming, databases and softwate design related with the finance world. HOW CRAZY I WAS!!

To be honest, I think it was the right choice because with the time passing I started loving more programming than financial markets. So that, on my way back to Spain y decided to focus and improve my programming skills to become an Full Stack Web Developer.

In order to get this goal, I am working hard, doing some online courses in 'Udemy', tutorials in 'The New Boston' and projects in 'FreeCodeCamp' to post them into my portfolio web page.

My dissertation project was based on the development of a financial website where the users can watch stock data retrieve from ‘Yahoo’s API’, perform technical analysis in charts provided by ‘Tradingview’, and build and monitor his own portfolio. This project has given me strong knowledge in website development, gaining experience in both front end and back end.

Once I ended up the Master, I did a internship at 'Shimmy Lingerie' in London for 3 months as a Junior Back End Developer. My functions were to create the Rest Api with Express.js and Node.js, apart from creating the Database with MongoDB using Mongoose to model the data.

After returning to Spain I worked as Junior Web Developer at 'La vida en Led' for 3 months. This website was developed with Prestashop and my functions were to use the 'Administrator Interface' in order to maintain the website, update all the existing products and upload new goods.

Forementioned my objective is to become a Full Stack Web Developer and for that to happend I am constantly learning new languages, framworks and tools, from building and maintaining websites to playing with APIs and databases in order to build modern web applications.

My strength is with Html 5 and Css 3 and Bootstrap, along with Javascript, but I also enjoy working on the back end and try to expand my skills learning new frameworks like Angular, languages like Php and databases like MySql o MongoDB.
